# KoreForge.Web

ASP.NET Core REST API building blocks for KoreForge applications.

## Packages

- **KoreForge.Web.RestApi.Abstractions** — shared contracts, options, and extension points
- **KoreForge.Web.RestApi.Observability** — observability helpers
- **KoreForge.Web.RestApi.Persistence** — persistence helpers
- **KoreForge.Web.Authorization** — role semantics engine for ASP.NET Core
